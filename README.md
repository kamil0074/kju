aka kompilatory-2019-1

## Quick start

### Manual

Using `dotnet` command.

Eg. on Linux.

#### Run application

```
$ cd src/KJU.Application
$ dotnet run
```

#### Build core

```
$ cd src/KJU.Core
$ dotnet build
```

#### Test

```
$ cd src/KJU.Tests
$ dotnet test /p:CollectCoverage=true
```

### Visual Studio

- Install .Net Core 2.1 on Visual Studio

- Open `KJU.sln`

- Use Visual Studio features
